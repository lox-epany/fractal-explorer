# Техническое задание: Fractal Explorer

## 1. Описание проекта
Интерактивное приложение для визуализации и исследования фракталов (Мандельброт, Жюлиа).

## 2. Функциональные требования
- Визуализировать множество Мандельброта
- Визуализировать множество Жюлиа
- Интерактивное масштабирование (зум)
- Сохранение параметров в БД
- Галерея сохранённых фракталов
- Экспорт в PNG

## 3. Нефункциональные требования
- Быстрое вычисление (C++ параллелизм)
- Отзывчивый UI (многопоточность PyQt6)
- Красивое отображение (800x600)

## 4. Архитектура
- **UI слой:** PyQt6 (Python)
- **Вычисления:** C++ (DLL через ctypes)
- **База данных:** SQLite
- **Потокизация:** QThread для долгих операций

## 5. Технологии
- Python 3.11
- PyQt6 6.6.1
- C++17
- SQLite3
- CMake для компиляции C++
