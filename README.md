# Fractal Explorer

Интерактивное приложение для визуализации и исследования фракталов (Множество Мандельброта и Жюлиа).

## Возможности

- Визуализация множества Мандельброта
- Визуализация множества Жюлиа  
- Интерактивное масштабирование и панорамирование
- Сохранение параметров фракталов в базу данных
- Галерея сохранённых фракталов
- Экспорт в PNG

## Технологии

- **Frontend**: PyQt6 (Python)
- **Вычисления**: C (библиотека через ctypes)
- **База данных**: SQLite3
- **Многопоточность**: QThread

## Установка и запуск

### Требования
- Python 3.11+
- PyQt6
- Компилятор C (GCC/Mingw)

### Установка зависимостей
```bash
pip install -r .\requirements.txt
```

### Компиляция C библиотеки
```bash
# Перейти в папку cpp/
cd cpp

# Компиляция для Windows
gcc -shared -o ../lib/mandelbrot.dll mandelbrot.c

# Компиляция для Linux  
gcc -shared -fPIC -o ../lib/libmandelbrot.so mandelbrot.c
```

### Запуск
```bash
# пока просто вручную запускайте main_window.py
python src/ui/main_window.py
```
